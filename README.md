# SmartFurnish

SmartFurnish is a full-stack furniture web application where users can browse products, add items to cart, and place orders.

## Features
- User login system
- Browse furniture products
- Add to cart
- Order tracking
- Payment page
- Admin product upload

## Technologies Used
Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Node.js
- Express.js
- MongoDB

## Project Structure
frontend/ – UI pages  
backend/ – API and server logic  
screenshots/ – UI preview images  

## How to Run

1. Clone the repository
2. Install backend dependencies

npm install

3. Start server

node src/server.js

4. Open frontend/index.html
